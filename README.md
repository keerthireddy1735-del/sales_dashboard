# sales_dashboard
This project analyzes 500 rows of sales data and visualizes key metrics through an interactive Excel dashboard.
This highlights:
- Total Sales  
- Total Quantity Sold  
- Total Orders  
- Sales by Month  
- Sales by Region  
- Sales by Product  
- Sales by Category
The Dashboard includes "interactive Slicers: for:
-Product
-Category
-Region
Users can filter instantly to see updates insights.
Tools and Techniques Used
-Microsoft Excel
-Pivot Tables
-Pivot Charts
-Data Cleaning & Preparation
-KPI Cards
Insights Visualized
- Monthly Sales Trend – Line chart  
- Regional Performance – Bar chart  
- Top Products – Column chart  
- Category-wise Sales – Visual comparison
 Key Learnings
- Designing professional dashboards
